## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Verdant31/Worldtrip.git
```

```
# Acesse a pasta da aplicação
$ cd Worldtrip

# Instale as dependências
$ yarn

# Inicie a aplicação
$ yarn start

```
